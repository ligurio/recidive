About

Récidive store test results.

The source code for Récidive is available on GitHub.

Add new test results

Récidive supports three the most popular test results formats:

  • TAP (Test Anything Protocol)
  • SubUnit
  • JUnit

To add a test result to Récidive, open Upload page, choose desired file in supported format and press Upload button.

For batch upload you may use curl: 

curl -X POST -F "data=@report.subunit" http://{{.}}/upload
